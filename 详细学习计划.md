### Day 13：PID算法原理

**目标**：理解PID控制理论

**学习内容**：
- PID三个参数的物理意义
- 位置式PID vs 增量式PID
- PID调参方法
- 积分饱和与抗饱和

**代码阅读**：

| 文件 | 关键函数 |
|------|----------|
| `BSP/PID/pid.c` | `PID_Init()`, `PID_SetTarget()`, `PID_Calculate()`, `PID_Reset()` |

**关键公式**：
```
output = Kp × error + Ki × integral + Kd × (error - last_error)
error = target - actual
integral += error（积分累加）
```

**本项目PID参数**：
```
Kp = 14.0    （比例系数：提高响应速度）
Ki = 1.65    （积分系数：消除稳态误差）
Kd = 0.00    （微分系数：本项目设为0，因为编码器噪声大）
输出限幅：0~1000（对应PWM占空比）
```

**动手实践**：
1. 阅读pid.c源码
2. 手写一个简化版PID函数
3. 用串口+上位机观察PID响应曲线

**知识点清单**：

- [ ] 为什么本项目Kd=0
- [ ] 输出限幅的意义
